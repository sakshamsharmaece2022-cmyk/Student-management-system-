# Student Management System (Java)

A simple Java console-based project demonstrating core Java and OOP concepts.

## Features
- Add student
- View all students
- Uses OOP principles (Class, Object, Encapsulation)
- Menu-driven console application

## Project Structure
src/
- Main.java
- Student.java
- StudentService.java

## How to Run
1. Clone the repository
2. Open the project in any Java IDE (IntelliJ / Eclipse / VS Code)
3. Run Main.java

## Technologies Used
- Java
- OOP
